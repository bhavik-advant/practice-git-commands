# Git Practice Project

This is a simple boilerplate project to practice git commands.

## Project Structure
- `index.html` - Main HTML file
- `styles.css` - Stylesheet
- `script.js` - JavaScript functionality
- `README.md` - This file

## Git Commands to Practice

1. **Initialize repository:**
   ```
   git init
   ```

2. **Check status:**
   ```
   git status
   ```

3. **Stage files:**
   ```
   git add .
   git add <filename>
   ```

4. **Commit changes:**
   ```
   git commit -m "Your message here"
   ```

5. **View commit history:**
   ```
   git log
   git log --oneline
   ```

6. **Create branches:**
   ```
   git branch <branch-name>
   git checkout -b <branch-name>
   ```

7. **Switch branches:**
   ```
   git checkout <branch-name>
   ```

8. **Merge branches:**
   ```
   git merge <branch-name>
   ```

9. **View differences:**
   ```
   git diff
   ```

10. **Undo changes:**
    ```
    git restore <filename>
    git reset HEAD <filename>
    ```

## Getting Started
1. Navigate to this directory
2. Run `git init` to initialize a git repository
3. Make changes to the files
4. Use `git add` and `git commit` to track changes
5. Try creating branches and merging them

Happy practicing!
